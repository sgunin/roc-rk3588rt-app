Detailed Description

Enumerate frame grabbers, create handles, and open frame grabbers.

Function Documentation

◆ MV_CC_EnumInterfaces()

MV_CAMCTRL_API int __stdcall MV_CC_EnumInterfaces ( IN unsigned int  nTLayerType,
IN OUT MV_INTERFACE_INFO_LIST *  pInterfaceInfoList 
)

Enumerate frame grabbers.

Parameters
nTLayerType[IN] Frame grabber interface type. See CameraParams.h for parameter definition.
Frame Grabber Interface Definition value Description
MV_GIGE_INTERFACE 0x00000001 GigE Vision frame grabber
MV_CAMERALINK_INTERFACE 0x00000004 Camera Link frame grabber
MV_CXP_INTERFACE 0x00000008 CoaX Press frame grabber
MV_XOF_INTERFACE 0x00000010 XoF Link frame grabber
pInterfaceInfoList[IN][OUT] Frame grabber list
Returns
Return MV_OK for success, and return corresponding error code for failure.
Examples
Events_Interface.cpp, InterfaceAndDeviceDemo.cpp, InterfaceDemo.cpp, and QuickSoftwareTrigger.cpp.
